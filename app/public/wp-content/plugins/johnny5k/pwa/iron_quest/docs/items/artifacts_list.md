# Johnny5k: IronQuest - Artifacts List

## Format

Artifacts include relics, charms, seals, idols, crowns, rings, and other special objects.

Each artifact includes:

- Tier
- Base cost in gold
- Artifact type
- Intended class or universal use
- Functional description
- Detailed visual description for AI image generation
- Gameplay modifier
