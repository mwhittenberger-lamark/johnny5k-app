## What This System Does (Critical)

This flow:

- Uses rest time for story
- Makes every set meaningful
- Connects effort to narrative
- Builds tension across the workout
- Creates replayability

## Key Design Rules (Lock These)

### 1. Every Set = Micro Outcome

- Performance affects story tone and HP

### 2. Every Exercise = Encounter

- New story beat
- New choice

### 3. Dice Drives Story Direction

- Not just success or failure
- But how things unfold

### 4. HP Loss Always Happens

- Reinforces effort = cost
- Bad rolls amplify it

### 5. Story During Rest Only

- Never interrupt lifts
- Enhances pacing naturally
